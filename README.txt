Description
===========
Ubercart Pasargad Bank Paypaad Payment provides Ubercart 2.x a new "Payment Method" to process its online payments
through the Paypaad Payment (http://paypaad.ir/) gateway.

Requirements
============
  - Paypaad 9-digit merchant code.
  - Paypaad 10-digit terminal code.
  - Private key file provided by Paypaad.

Dependencies
============
  - Paypaad Payment API (http://github.com/sepehr/paypaad-api)

Installation
============
  - Enable the module as usual.
  - Configure settings at admin/store/settings/payment/edit/methods.
